Odd Ceaser Documentation

About

The Odd Ceaser Encryption and Decryption Tool is a command-line application written in JavaScript that uses an algorithm similar to the Vigenere Cipher to encrypt and decrypt messages. The tool accepts both files and text as input.

A cipher is a method of transforming text to obscure its meaning, making it unreadable to anyone without the decryption key. Ciphers have been used throughout history to protect information from unauthorized access. The inspiration for this project came from a random computer science class in Sixth form and a book called "The Code Book: The Secret History of Codes and Code-Breaking" by Simon Singh. You can find more details in the README file on GitHub.

Hot Keys

  • `k`
    : This will kill the program at any stage.
  • `r`
    : This will restart the program at any stage.

Schema

The Odd Ceaser Cipher draws inspiration from a variety of ciphers, ranging from the simple Caesar cipher to the established Vigenere cipher, as well as more complex corporate ciphers like Lucifer or DES. The algorithm is suitable for basic encryption tasks and can run efficiently on most devices without requiring significant processing power.
